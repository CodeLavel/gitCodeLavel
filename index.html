เกี่ยวกับเรา

โค้ดลาเวล (CodeLavel) เว็บไซต์ที่จะให้ความรู้แก่ทุกท่าน ไม่ว่าจะเป็นเกี่ยวกับการเขียนโปรแกรมเบี้องต้น การทำเว็บไซต์เบื้องต้น Front-end , Back-end รวมถึงเทคนิคต่างๆอีกมากมาย พร้อมที่จะช่วยทำให้ทุกท่านเข้าใจในและแก้ไขปัญหาที่พบเจอ

รายละเอียด

Web Development

ผู้ที่มีหน้าที่ในการพัฒนาระบบ อาจจะมองได้ในส่วนของการ Research การค้นคว้าวิจัย เทคโนโลยี่ใหม่ ๆ ในการประมวลผลเว็บไซต์ ส่วน Web Programmer คือผู้ที่มีความเชี่ยวชาญในด้านการเขียนโปรแกรมทางคอมพิวเตอร์ ซึ่งมีความสามารถเฉพาะทางเพิ่มเติมในด้านการจัดทำเว็บไซต์ด้วย ทั้งนี้ในส่วนของงานที่เกี่ยวข้อง ยังมี Web Designer คือผู้ที่ทำงานด้านการออกแบบ งานกราฟฟิค การออกแบบเว็บไซต์ ให้ตรงความต้องการ ตอบสนองความต้องการในการใช้งาน โดยงานส่วนนี้ก็เปรียบได้เหมือนกับ มัณฑนากร ที่มีหน้าที่ออกแบบตกแต่งภายในร้านค้าให้มีความสวยงามนั่นเอง

Front-end, Back-end

Front-end สำหรับสาย Developer แล้วก็คือ หน้าบ้าน หรือจะเรียกให้ดูดีมีระดับก็คือ User Interface (UI) ยกตัวอย่างเช่น เฟสบุ๊ค หน้าแรกเฟสบุ๊ค หน้าเข้าสู่ระบบเฟสบุ๊ค หรือ แม้แต่ Google ที่เราใช้ค้นหากันนั้น ก็ล้วนจะเป็น Front-end ซึ่งความสำคัญของ Front-end ส่วนนี้จะเป็นส่วนที่แสดงหน้าตาเว็บไซต์ให้ผู้ที่เข้ามาชม ได้เห็นถึงการออกแบบซึ่งจุดนี้จะเป็นจุดที่ดึงดูด และทำให้ผู้ที่เข้ามาดูในเว็บไซต์เราสนใจในสิ่งที่เรานำเสนอนั้นเอง
Back-end สำหรับสาย Developer จะเรียกกันว่า หลังบ้าน เป็นส่วนที่เอาไว้จัดการข้อมูลระบบต่างๆที่มีภายในเว็บไซต์ ถ้าจะยกตัวอย่างเอาอย่างที่เข้าใจก็เหมือนกับการสร้างบ้าน ก่อนที่จะมาเป็นบ้านสวยๆต้องผ่านกระบวนการอะไรมาบ้าง เช่นการลงแบบเหล็ก หล่อปูน เสร็จคือการตกแต่ง และทาสี ในส่วนนั้นก็จะเป็นหน้าบ้าน แต่ถ้ามองในสายงาน Developer แล้วก้อ Back-end คือการจัดการฐานข้อมูล การจัดการโครงสร้างของเว็บไซต์

Graphic Design

ไม่ใช่แค่การออกแบบให้สื่อสารได้อย่างครบถ้วนเพียงอย่างเดียวจะเพียงพอ การพัฒนางานออกแบบให้ทันการเปลี่ยนแปลงของผู้ใช้งานก็เป็นสิ่งที่จำเป็น เพราะงาน Graphic Design เป็นงานที่ทำงานกับความรู้สึกและการรับรู้ของคน ซึ่งสิ่งเหล่านี้เปลี่ยนไปตลอดเวลา ไม่เคยเหมือนเดิมในแต่ละวัน วันนี้กลุ่มลูกค้าชอบที่จะกดซื้อสินค้าจากรูปที่เห็นส่วนลดชัดๆ เดือนต่อมาอาจจะไม่อยากกดแล้วเพราะรู้สึกถูกยัดเยียดขายมากเกินไป แต่หันไปซื้อสินค้าจากรูปโฆษณาที่สื่อสารแบบอื่นแทน ดังนั้น Graphic Designer จะเก็บข้อมูลอยู่ตลอด เพื่อพัฒนาให้งานออกแบบสอดคล้องกับจุดประสงค์แรกเริ่มของแคมเปญ และแสดงประสิทธิภาพได้สูงสุดทันพฤติกรรมผู้ใช้งานอยู่เสมอ
